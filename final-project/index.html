Summary:

We will create a pathtracer in C++ to simulate lip glosses on human tissue (potentially building off of Homework 3 if possible). We intend on creating a layered material system and changing parameters such as dielectric roughness (shine) and volumetric thickness/pooling (viscosity) to simulate various kinds of glosses.

Problem Description:

The main problem we intend to solve is how we can build a material system with a dielectric coating over an organic base (lip gloss over human lips) where we can dynamically change variables for shine and viscosity in order to simulate different consistencies of lip gloss. For example, a more high viscosity lip gloss would pool inside the crevices of the lips leading to a much smoother layer than the initial lip mesh. Since the gloss is thicker, it’ll create a smooth shell over the lips that will have a coherent reflection, creating a glassy look. Meanwhile, a lower viscosity lip gloss would create a much thinner even coating more similar to the initial lip mesh that will not be as perfectly smooth as the thicker gloss. As a result, this thinner gloss will create a dispersed reflection, giving a more moist/wet look.

In order to tackle the problem discussed above, we need to implement a layered BDRF, or “Clearcoat” model. Specifically for the specular glossy layer that the lip gloss creates, we plan on using the Cook-Torrance microfacet BDRF, and for the normal lip layer, we plan on using BSSRDF (Bidirectional Scattering Surface Reflectance Distribution Function) to calculate the subsurface scattering of light bouncing through the semi-opaque lips. We also need to consider energy conservation, as the light hitting the base lip layer must be reduced by the amount of light reflected in the lip gloss layer. (We will handle lip gloss color tints by multiplying the base layer by the transmittance color calculated using the Beer-Lambert law and then add it to the specular gloss reflection.)

Goals and Deliverables:

Our project aims to answer the question of how we can model light reflecting from various viscosities of lip gloss on human skin. In a similar vein, we aim to tackle how varying dielectric roughness mathematically and visually affects the “shine” of lip gloss when rendering.

For performance metrics, we will primarily focus on using images to show our results. Specifically, we can use a visual matrix to isolate and demonstrate the different results that come from changing dielectric roughness and thickness/viscosity parameters. We could also compare our rendered images to real-life images of lip gloss in similar environmental lighting.

Target Goals:
We aim to implement a layered BSDF into our pathtracer with a base layer for skin (lips) and a dielectric layer for the gloss. We will also make sure there are three adjustable parameters: one for dielectric roughness to adjust the shine/glossiness, one for thickness to adjust the viscosity, and one for color/saturation.

Reach Goals:
Some goals we hope to achieve include adding geometry-driven pooling instead of having uniform thickness for the gloss, which will allow the gloss to settle into cracks/crevices of the lip mesh, adding BSSRDF to the lip tissue to make it look more realistic and skin-like, and adding iridescence to the gloss layer to simulate glitter, which is sometimes found in lip gloss products. Additionally, if we have extra time, we were thinking about simulating a ferromagnetic lip gloss.
